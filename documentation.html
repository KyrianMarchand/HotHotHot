Présentation de l'équipe

L'équipe est composée de 5 membres : Thibault, Kyrian, Cyril, Killian et Lise.
Pour plus d'efficacité, nous nous sommes répartis les tâches comme suit :
Thibault : Mise en place du serveur, HTML / CSS / Javascript
Kyrian : Mise en place du Github, HTML / CSS / Javascript
Cyril : HTML / CSS / Javascript
Killian : Rédaction de la documentation, HTML / CSS
Lise : Rédaction de la documentation, HTML / CSS



Les différents outils utilisés

Pour la programmation, nous avons utilisé le logiciel Visual Studio Code.
Pour le partage des fichiers, nous avons utilisé GitHub. ainsi que Google Drive.
Pour l'hébergement du site, nous avons utilisé un hébergeur gratuit, Alwaysdata.
Pour la réalisation du diagramme de Gantt, nous avons utilisé le site web Lucidchart.
